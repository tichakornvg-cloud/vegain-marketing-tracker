🔧 ตั้งค่า Firebase ก่อนใช้งาน

กรอก Firebase config ด้านล่างเพื่อเชื่อมต่อฐานข้อมูลกลาง — ทำครั้งเดียวแล้วทีมใช้ร่วมกันได้เลย

1. สร้าง Firebase Project ไปที่ console.firebase.google.com → "Add project" → ตั้งชื่อ vegain-marketing → ปิด Google Analytics → Create project
2. สร้าง Realtime Database เมนูซ้าย → "Realtime Database" → "Create database" → เลือก Singapore → Start in test mode → Enable
3. ก็อป Config Project Settings (⚙️) → "Your apps" → Web app (</>) → Register app → ก็อป firebaseConfig แล้วกรอกด้านล่าง

Config จะถูกบันทึกใน browser — ทีมแต่ละคนต้องกรอกครั้งเดียวเท่านั้น
